Content

Depends on ID "content-body"

All application code should beplaced inside a div with ID content-body, and wrapped in a ".container"

Code


		<div id="content-body">    
				<div class="container">
		        <h1 class="page-header">Beheren: Activiteiten</h1>
				</div>
		</div>
